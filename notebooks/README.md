# Notebooks Directory

This directory contains Jupyter notebooks for data exploration, analysis, and experimentation.

## Structure

### `exploratory/`
Initial data exploration notebooks for each dataset. Use these to:
- Understand data structure and quality
- Identify patterns, outliers, and missing values
- Generate initial visualizations

**Naming convention:** `01_explore_[dataset_name].ipynb`

### `analysis/`
Focused analysis notebooks for specific research questions or hypotheses:
- Statistical analysis
- Correlation studies
- Trend analysis
- Comparative studies

**Naming convention:** `[dataset_name]_[analysis_type].ipynb`

### `experiments/`
Experimental notebooks for testing ideas, prototyping features, or trying new approaches:
- Model prototypes
- Visualization experiments
- Data processing experiments
- Feature engineering tests

**Naming convention:** `experiment_[description]_[date].ipynb`
