
# Data Directory

This folder contains all datasets used in the project.

---

## Structure

- Each dataset is organized in its own subfolder.
- Every subfolder includes a `README.md` file describing:
  - The dataset's contents
  - The data source
  - Any relevant notes or preprocessing steps

## Usage

To explore a dataset, navigate to its subfolder and refer to its `README.md` for detailed information.

---

**Note:**  
Keep this directory organized by ensuring each new dataset is placed in a separate subfolder with proper documentation.
